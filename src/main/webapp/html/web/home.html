    胜利分院
    为了缓解总院患者停车难、就诊拥挤等不便，山西省中医院于2012年年初在太原市杏花岭区胜利街160号新增设院址成立山西省中医院胜利分院，胜利分院隶属山西省中医院，诊疗方式以门诊为主，是省、市基本医疗保险、铁路医保及农村合作医疗定点机构，医疗资源与总院共享，出诊专家均为山西省中医院副高职称以上专家，汇集山西省中医界名老专家和学术权威。特别方便北城地区群众对中医药医疗的需求。除日常诊疗外分院还提供健康体检服务。
    查看详细
    和平分院
    山西省中医院和平分院系山西省卫生厅批准设立的中医医疗机构，是山西省中医院的下属机构，位于万柏林区和平南路336号。和平分院开设六年来，门诊量呈快速增长的态势，年门诊量八万余人次，满足了河西百姓看中医，求名中医的需求。山西省中医院百余名中医及中西医结合专家在分院出诊，为各专业疑难病症病人提供有效治疗技术。影响力波及方圆百里。

山西省中医院动员部署“打击收受红包加强行风建设”专项行动

2021-07-27

7月26日下午，山西省中医院召开“打击收受红包，加强行风建设”专项行动动员部署会议，医院党委领导班子成员以及全体中层干部参加。会议由院长刘光珍主持。会上，山西省中医院纪委书记郭瑞对“打击收受红包，加强行风建设”专项行动进行了详细全面的安排部署。他表示，医院将以严禁收受红包，严厉打击收受回扣，纠正诊疗服务乱象，改善医疗服务态度以及优化完善医疗环境为重点，全面开展专项行动。全院职工要深刻认识行风建设的重要意义，要认真执行医院各项核心制度，规范医疗程序，依法执业。

省中医院 古籍保护，我们在行动

2021-07-27

7月23日，由中国古籍保护协会主办，山西省图书馆（山西省古籍保护中心）承办的“2021年中华古籍普查文化志愿服务行动 · 山西行”活动来到了山西省中医院。省中医院作为我省中医药最高研究机构，有着相对丰富的馆藏中医药古籍，省图书馆党委书记冀保胜感谢省中医院加入本次志愿活动，他认为，作为医疗机构加入开展古籍普查工作，一定会成为我省古籍保护工作的一大亮点。

程永平副市长一行莅临山西省中医院调研医疗废物无害化管理工作

2021-06-21

2021年6月18日，太原市副市长程永平、市政府副秘书长杨兴海、市生态环境局局长闫文斌一行莅临山西省中医院，实地调研我市医疗废物收集、贮存、转移及医疗废水处理有关情况。 在山西省中医院谭利国书记、李庭凯副院长、李辉副书记陪同下，程永平一行先后前往我院医疗废物贮存站、污水处理站进行了实地调研。谭利国书记就我院长态化疫情防控和污水处理、医疗废弃物管理等工作做了详细介绍。

山西省中医院党委召开巡视工作部署会

2021-02-25

2月24日上午，山西省中医院召开党委扩大会议，对巡视工作进行了专题部署。院领导班子全体成员及党委办、院办、人事科、纪检监察室、宣传部等相关职能部门负责人参加了本次会议。 党委书记谭利国主持会议，传达了中共山西省委巡视工作办公室下发的《关于省委巡视组对省中医药研究所党委开展专项巡视的通知》，对通知要求的思想准备、组织准备、会议准备、工作汇报、宣传准备、材料准备、保障准备七个方面进行了详细解读。会上，谭利国书记宣布了联络组组长及成员名单，并对联络接洽、材料准备、会议安排、保障工作、宣传工作等具体事项进行了安排部署。
