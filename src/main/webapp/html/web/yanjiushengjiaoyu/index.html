    胜利分院
    为了缓解总院患者停车难、就诊拥挤等不便，山西省中医院于2012年年初在太原市杏花岭区胜利街160号新增设院址成立山西省中医院胜利分院，胜利分院隶属山西省中医院，诊疗方式以门诊为主，是省、市基本医疗保险、铁路医保及农村合作医疗定点机构，医疗资源与总院共享，出诊专家均为山西省中医院副高职称以上专家，汇集山西省中医界名老专家和学术权威。特别方便北城地区群众对中医药医疗的需求。除日常诊疗外分院还提供健康体检服务。
    查看详细
    和平分院
    山西省中医院和平分院系山西省卫生厅批准设立的中医医疗机构，是山西省中医院的下属机构，位于万柏林区和平南路336号。和平分院开设六年来，门诊量呈快速增长的态势，年门诊量八万余人次，满足了河西百姓看中医，求名中医的需求。山西省中医院百余名中医及中西医结合专家在分院出诊，为各专业疑难病症病人提供有效治疗技术。影响力波及方圆百里。

山西省中医药研究院·山西省中医院成立于1957年，原名山西省中医研究所·山西省中医研究所附属医院。历经60余年的建设，目前已发展成为山西省集医疗、教学、科研于一体，具有鲜明中医药特色的综合性三级甲等中医院。医院设有26个病区、2个分院、东山科教园区。临床科室52个，医技及辅助科室19个。全院占地面积8.62万平方米，建筑面积8.04万平方米，总资产8.9亿元，现床位编制达1058张，开放床位1155张。

2019年6月6日下午，山西省中医药研究院2019届硕士研究生学位论文答辩会在我院综合楼圆满落幕。32名研究生参加了答辩，为期3天，分肿瘤消化组、妇科外科组、基础组等4场进行。郝旭亮副院长高度重视此次答辩工作，对答辩委员会成员名单严格审核，并请专家严格评审学生论文，为便于交流，提高我院硕士论文水平，所邀请专家均为外院、省内乃至国内优秀专家。会议由研究生部主任谭宝主持。
